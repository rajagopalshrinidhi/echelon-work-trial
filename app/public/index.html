API docs & usage 
Your site is served as static files — any HTML/CSS/JS works, no build step.

To save and load shared data, use:
  // save
  fetch('/_api/data/<your-site-name>', {method:'POST', headers:{'Content-Type':'application/json'}, body: JSON.stringify({...})})
  // load
  const entries = await fetch('/_api/data/<your-site-name>').then(r => r.json())

To upload a file and get a URL back:
  const fd = new FormData(); fd.append('file', fileInput.files[0]);
  const {url} = await fetch('/_api/files/<your-site-name>', {method:'POST', body: fd}).then(r => r.json())

Do not use localStorage/sessionStorage for anything that needs to be shared or durable —
all sites share the same browser origin, so storage keys can collide across sites.
Use the data API above instead.
